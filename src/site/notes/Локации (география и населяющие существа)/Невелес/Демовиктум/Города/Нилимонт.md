# Приорат Иерехона
Большое здание, похожее на скалистый выступ. Грубо отесаный желтоватый камень. Постройка напоминает ступенчатый акрополь, на вершине которого находится галерея с колоннами. На самом верху, на крыше здания из камня вырезана голова мрачного существа с горящей головой. Факел этой головы освещает округу.
